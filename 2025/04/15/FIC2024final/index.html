复现时遇到

TASK ERROR: max 2 vcpus allowed per vm on this node

步骤：

  1. 打开配置文件： vi /usr/share/perl5/PVE/QemuServer.pm
  2. 搜索并注释掉以下行： # die "MAX $allowed_vcpus vcpus allowed per VM on this node\n" if ($allowed_vcpus < $maxcpus);
  3. 重启pvedaemon服务： systemctl restart pvedaemon
